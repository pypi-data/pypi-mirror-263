# DBRepo Python Library

Official client library for [DBRepo](https://www.ifs.tuwien.ac.at/infrastructures/dbrepo/), a database repository to
support research.

## Installing

```console
$ python -m pip install dbrepo
```

This package supports Python 3.10+.
